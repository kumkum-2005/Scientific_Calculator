# Scientific_Calculator-python

A beautiful, fully-functional Scientific Calculator built using Python. This calculator performs basic arithmetic operations along with advanced scientific calculations in a simple and user-friendly interface.

# Features

#📱 Simple Calculator
      Basic arithmetic operations (+, -, ×, ÷)
      Percentage calculations
      Decimal support
      Delete/Backspace functionality
      Live calculation history - See your recent calculations as you work
      Clear history option
